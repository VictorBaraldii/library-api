# Library API

API REST para gerenciamento de livros e usuários usando Spring Boot, Maven e Docker.

---

## Tecnologias

- Java 17
- Spring Boot
- Spring Data JPA
- Maven
- Docker
- Swagger (documentação)
- PostgreSQL / banco configurável via env

---

## Funcionalidades

A API permite:

- Criar, listar, buscar, atualizar e excluir livros
- (Se tiver) gerenciamento de usuários
- Documentação Swagger UI
